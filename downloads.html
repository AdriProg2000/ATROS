Descarga la última versión de ATROS para tu ordenador.

Comandos de instalacion (Fedora/Mageia):

cd Descargas

unzip atros_v(version).zip

cd atros_v(version).zip

sudo dnf install gcc

sudo dnf install gcc-g++

./boot.sh
